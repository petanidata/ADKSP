# Panduan Instalasi Orange Data Mining

Panduan ini disusun untuk mahasiswa mata kuliah Analitika Data Keuangan Sektor Publik agar dapat memasang Orange Data Mining dengan mudah. Ikuti bagian yang sesuai dengan perangkat yang digunakan.

Unduh Orange dari laman resmi berikut:

[https://orangedatamining.com/download/](https://orangedatamining.com/download/)

Pilih versi standalone, bukan portable.

## 1. Instalasi Orange Data Mining di Windows

### A. Persiapan

Sebelum mulai, pastikan perangkat Windows sudah terhubung ke internet dan Anda memiliki izin untuk memasang aplikasi. Versi standalone Orange tidak memerlukan instalasi manual tambahan setelah file diunduh.

### B. Langkah instalasi

1. Buka halaman unduhan Orange di [https://orangedatamining.com/download/](https://orangedatamining.com/download/).
2. Pada bagian Windows, pilih **Standalone installer (default)**.
3. Unduh file installer bernama **Orange3-3.40.0-x86_64.exe**.
4. Setelah unduhan selesai, buka file tersebut.
5. Jika muncul peringatan keamanan Windows, pilih **Run anyway** atau **Yes** jika Anda yakin file berasal dari situs resmi Orange.
6. Ikuti petunjuk instalasi sampai selesai.
7. Jika diminta memilih lokasi instalasi, gunakan lokasi default agar lebih mudah.
8. Setelah instalasi selesai, buka **Orange** dari menu Start atau dari shortcut yang tersedia di desktop.

### C. Setelah instalasi

Saat Orange pertama kali dibuka, tunggu beberapa saat hingga aplikasi selesai memuat. Jika Orange berhasil terbuka, berarti instalasi sudah selesai.

### D. Jika mengalami kendala

1. Pastikan file yang diunduh adalah installer **.exe**, bukan file **.zip**.
2. Jika instalasi gagal, coba jalankan file installer dengan klik kanan lalu pilih **Run as administrator**.
3. Jika masih gagal, pastikan antivirus atau kebijakan kampus tidak memblokir pemasangan aplikasi.

## 2. Instalasi Orange Data Mining di Mac

### A. Persiapan

Pastikan Mac Anda terhubung ke internet dan memiliki ruang penyimpanan yang cukup. Di Mac, Orange tersedia dalam format **.dmg**.

### B. Menentukan jenis Mac

Sebelum mengunduh, pastikan jenis prosesor Mac Anda:

1. Klik logo Apple di pojok kiri atas.
2. Pilih **About This Mac**.
3. Periksa bagian **Chip** atau **Processor**.
4. Jika tertulis **Apple**, pilih installer untuk **Apple silicon**.
5. Jika tertulis **Intel**, pilih installer untuk **Intel**.

### C. Langkah instalasi

1. Buka halaman unduhan Orange di [https://orangedatamining.com/download/](https://orangedatamining.com/download/).
2. Pada bagian macOS, pilih installer yang sesuai dengan perangkat Anda.
3. Unduh file **.dmg** yang sesuai:
	- **Orange3-3.40.0-Python3.12.10-arm64.dmg** untuk Mac Apple silicon.
	- **Orange3-3.40.0-Python3.12.10-x86_64.dmg** untuk Mac Intel.
4. Setelah unduhan selesai, buka file **.dmg** tersebut.
5. Seret ikon **Orange** ke folder **Applications** jika diminta.
6. Buka Orange dari folder **Applications** atau melalui Launchpad.
7. Jika muncul peringatan keamanan dari macOS, pilih **Open** untuk melanjutkan.

### D. Setelah instalasi

Saat Orange pertama kali dibuka, macOS mungkin meminta konfirmasi tambahan. Ikuti instruksi di layar sampai aplikasi berhasil terbuka.

### E. Jika mengalami kendala

1. Pastikan Anda memilih installer yang sesuai dengan chip Mac.
2. Jika Orange tidak bisa dibuka, coba buka dari folder **Applications**.
3. Jika muncul pesan keamanan, buka **System Settings** lalu periksa bagian **Privacy & Security** untuk mengizinkan aplikasi.

## 3. Catatan untuk Mahasiswa

1. Gunakan versi standalone resmi agar instalasi lebih mudah.
2. Jangan memilih versi portable karena panduan ini disusun untuk instalasi standar.
3. Setelah Orange terbuka, mahasiswa dapat langsung menggunakannya untuk praktik analitika data.

## 4. Ringkasan Singkat

- **Windows**: unduh file **.exe** standalone, lalu jalankan installer.
- **Mac**: unduh file **.dmg** sesuai jenis chip, lalu pindahkan Orange ke **Applications**.
- Sumber unduhan resmi: [https://orangedatamining.com/download/](https://orangedatamining.com/download/).
